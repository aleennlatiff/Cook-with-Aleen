# Cook with Aleen 🍳

## About This Project
Hello! This is my midterm project for the Internet and Web Programming class. I built a recipe website to practice and apply what we learned about HTML, CSS, and JavaScript.

Instead of writing every single recipe directly into the HTML file (which would be repetitive), I wanted to challenge myself. I learned how to use a JavaScript array as a "mock database".
 This was a great way for me to practice DOM manipulation and make the website interactive without needing to refresh the page.

## Features I Built
* **JavaScript Data:** I stored the recipe details (images, titles, difficulty) in a JS array and used a function to display them on the screen automatically.
* **Search & Filter Buttons:** If you type in the search bar or click a category button (like "Asian" or "Mexican"), my JavaScript code filters the array and shows only the matching recipes.
* **Dark Mode:** I created a light/dark mode switch. I also learned how to use `localStorage` so the browser remembers your choice even if you close the page!
* **Hero Slider:** I built a simple image slider at the top of the page with Next/Previous buttons.

## Tools Used
I only used the basic front-end tools we covered in our lessons:
* HTML5 (for the structure)
* CSS3 (for the styling and responsive design)
* Vanilla JavaScript (for the logic and interactions)

## How to View the Project
It is very easy to run because there is no complicated backend:
1. Download or clone these files.
2. Double-click on the `index.html` file.
3. It will open directly in your web browser!
